### Website for [SavvyShots Photography](http://savvyshots.photography)

### Setup

- Fork/Clone
- Requires http://bower.io/
    - run `bower install`
    - run `npm build` -> make sure there are no errors
- run `middleman server`
- visit `locahost:4567`
- Party!
